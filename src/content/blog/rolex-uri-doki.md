<!-- ===== H2-1（まず確認すること） ===== -->
<h2>ロレックスを売る前に、まず確認すること</h2>

<!-- 💡 解決策（結論） -->
<div style="background:#FFF8E1;border-left:4px solid #C8923A;padding:18px 22px;margin:24px 0;border-radius:4px;">
<p style="margin:0 0 8px 0;"><strong style="color:#C8923A;">💡 解決策（結論）</strong></p>
<ul style="margin:0;padding-left:1.4em;line-height:1.85;">
<li><span style="color:#27AE60;">✅</span> <strong>型番（Ref）・購入時期・状態を把握しておく</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>付属品（保証書・箱・余りコマ）の有無を確認する</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>動作不良やキズは"直さず"そのまま伝えるだけでOK</strong></li>
</ul>
</div>

<p>売却をスムーズに、かつ有利に進めるために、まず手元の一本の情報を整理しておきましょう。査定士に「分かっている人だ」と伝わると、交渉も進みやすくなります。確認するのは次の3つです。</p>

<p><strong>① 型番(Ref)・購入時期・状態</strong>：保証書や裏蓋に記載された型番（例：126500LN）、いつ・どこで買ったか、動作やキズの状態をざっくり把握しておく。<strong>② 付属品の有無</strong>：保証書（ギャランティカード）・外箱・余りコマ・タグ・購入時のレシートがあるか。<strong>③ 不具合の有無</strong>：止まっている・ガラスが割れている・キズがある——こうした状態も、<strong>自分で直さず「現状のまま」伝えるだけ</strong>で大丈夫です。理由は次の章で詳しく説明します。</p>

<div style="background:#E0F2F1;border:4px solid #00695C;border-radius:8px;padding:18px 22px;margin:28px 0;">
<p style="margin:0 0 10px 0;"><strong style="color:#004D40;">💡 業界人ワンポイント</strong></p>
<p style="margin:0;line-height:1.85;">査定の現場で感じるのは、<strong>「型番とおおよその相場を知っている人ほど、適正額に近づく」</strong>ことです。完璧に調べる必要はありません。型番をスマホにメモして、後で出てくる「無料の一括査定」で今の相場感をつかんでおくだけで、現場で足元を見られにくくなります。</p>
</div>

<!-- ===== H2-2（オーバーホール不要） ===== -->
<h2>オーバーホール・研磨は不要｜むしろ価値が下がる</h2>

<!-- 💡 解決策（結論） -->
<div style="background:#FFF8E1;border-left:4px solid #C8923A;padding:18px 22px;margin:24px 0;border-radius:4px;">
<p style="margin:0 0 8px 0;"><strong style="color:#C8923A;">💡 解決策（結論）</strong></p>
<ul style="margin:0;padding-left:1.4em;line-height:1.85;">
<li><span style="color:#27AE60;">✅</span> <strong>売る前のオーバーホール費用は、査定アップ分を上回らない</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>研磨はケースが痩せて"逆に"価値が下がる</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>止まっている・割れている時計でも、現状のまま売れる</strong></li>
</ul>
</div>

<p>売る前にいちばんやりがちな失敗が、<strong>「きれいにしてから・直してから売ろう」</strong>です。気持ちは分かりますが、ロレックスに関しては<strong>これが最大の損</strong>になります。</p>

<p><strong>① オーバーホール（分解掃除）は費用倒れ</strong>：正規でのオーバーホールは数万〜10万円超かかりますが、その分が査定額に上乗せされて回収できることは、まずありません。買取店は自社の工房や安い業者価格で整備できるため、<strong>個人が高いお金をかけて整備する必要はゼロ</strong>です。</p>

<p><strong>② 研磨（ポリッシュ）は致命的</strong>：キズを消そうと磨くと、金属を削るため<strong>「ケース痩せ」</strong>が起きます。ラグが細くなり、本来シャープなエッジが丸くなり、防水性も落ちる。高級時計の世界で最も評価が高いのは、<strong>磨かれていない「ノンポリッシュ（未研磨）」のオリジナル状態</strong>です。細かいキズは、買取店が再販前にプロの基準で必要最小限だけ磨くので、<strong>あなたは何もしないのが正解</strong>です。</p>

<p><strong>③ 止まっていても、割れていても売れる</strong>：動かない・水没した・ガラスが割れた・ブレスがちぎれた——そんな状態でも、<strong>ロレックスは純正パーツそのものに価値がある</strong>ため、しっかり値段がつきます。諦めて捨てたり、高い修理代を払って直したりせず、ジャンクのまま査定に出してください。</p>

<div style="background:#fff;border:3px solid #3E2723;border-radius:10px;padding:20px 24px;margin:28px 0;">
<p style="margin:0 0 12px 0;font-weight:bold;color:#3E2723;font-size:1.05em;">📌 例外：この2つだけは"手元にあれば"プラス</p>
<ul style="margin:0;padding-left:1.4em;line-height:2;">
<li><strong>過去2年以内の正規整備の明細・保証書がある</strong>なら、新たに費用はかけず、その書類を<strong>一緒に提示</strong>（真正性の証明になる）</li>
<li><strong>アンティーク・ヴィンテージ（数十年前の古いモデル）は、絶対にオーバーホールに出さない</strong>。古い文字盤や針が現行品に交換されると、コレクター価値が大きく下がるため、手つかずのまま専門の鑑定士へ</li>
</ul>
</div>

<!-- ===== H2-3（高く売りやすくなる準備・軽め） ===== -->
<h2>高く売りやすくなる、ちょっとした準備</h2>

<!-- 💡 解決策（結論） -->
<div style="background:#FFF8E1;border-left:4px solid #C8923A;padding:18px 22px;margin:24px 0;border-radius:4px;">
<p style="margin:0 0 8px 0;"><strong style="color:#C8923A;">💡 解決策（結論）</strong></p>
<ul style="margin:0;padding-left:1.4em;line-height:1.85;">
<li><span style="color:#27AE60;">✅</span> <strong>柔らかい布で軽くホコリ・手垢を拭く程度でOK（研磨剤はNG）</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>保証書・箱・余りコマをそろえると査定が上がりやすい</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>付属品がなくても売れる｜ただしある方が確実に有利</strong></li>
</ul>
</div>

<p>「何もするな」と言いましたが、<strong>ダメージを与えない範囲の準備</strong>はプラスに働きます。やることは2つだけ。</p>

<p><strong>① 軽く拭く</strong>：柔らかいメガネ拭き（マイクロファイバー）で、ケースやガラスの手垢・ホコリを優しく拭き取る程度。溝の黒ずみは、毛先の柔らかい古い歯ブラシで軽く。<strong>研磨剤・歯磨き粉・水洗いは厳禁</strong>です。清潔感があると、査定士の心証が良くなり評価が上がりやすくなります。</p>

<p><strong>② 付属品をそろえる</strong>：保証書・外箱・余りコマ・タグを探して一緒に出す。特に<strong>保証書は査定に大きく効きます</strong>（次の章の準備にもなります）。付属品がなくても売れますが、ある方が確実に有利です。保証書がない場合の影響や探し方は、このあとの章で詳しく触れます。</p>

<!-- ===== H2-4（★売り時・メインKW） ===== -->
<h2>ロレックスの売り時はいつ？「季節×為替」で見極める</h2>

<!-- 💡 解決策（結論） -->
<div style="background:#FFF8E1;border-left:4px solid #C8923A;padding:18px 22px;margin:24px 0;border-radius:4px;">
<p style="margin:0 0 8px 0;"><strong style="color:#C8923A;">💡 解決策（結論）</strong></p>
<ul style="margin:0;padding-left:1.4em;line-height:1.85;">
<li><span style="color:#27AE60;">✅</span> <strong>需要期（3・6・12月）×円安 が重なる局面が狙い目</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>相場は「円安で上昇・円高で下落」とシンプルに連動</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>モデルごとにピークが違う｜サブマリーナは夏前(4〜5月)</strong></li>
</ul>

<p>「いつ売るのが得か」を決めるのは、昔は季節需要でしたが、いまは<strong>為替（円安・円高）の影響が非常に大きい</strong>。だから売り時は<strong>「季節 × 為替」の掛け算</strong>で見るのが正解です。</p>

<p>ちなみにロレックスは<strong>2026年1月にも正規店で定価改定（値上げ）</strong>が行われました。定価が上がると新品が買いにくくなり、中古の割安感が強まるため、<strong>相場全体が底上げ</strong>されやすくなります。そこに歴史的な円安も重なっている2026年は、売り手にとって追い風の局面と言えます。</p>

<div style="margin:24px 0;">
<p style="margin:0 0 6px 0;font-weight:bold;color:#3E2723;">Q. ロレックスの売り時はいつですか？</p>

<p style="margin:0 0 18px 0;">A. いいえ。オーバーホール費用が査定アップ分を上回ることはまずなく、費用倒れになります。研磨はケースが痩せて逆に価値が下がります。止まっていても割れていても売れるので、現状のまま出すのが正解です。</p>

<p style="margin:0 0 18px 0;">A. 売れます。ロレックスは純正パーツ自体に価値があるため、不動・水没・ガラス割れでも値段がつきます。自分で直さず、現状のまま査定に出してください。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#3E2723;">Q. 査定前に自分でキズを磨いてもいいですか？</p>
<p style="margin:0 0 18px 0;">A. 絶対にやめてください。研磨剤や歯磨き粉での自己研磨は仕上げを傷め、大幅な減額になります。柔らかい布で手垢やホコリを軽く拭く程度に留めましょう。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#3E2723;">Q. 高額なので持ち運びが不安です。どう売ればいい？</p>
<p style="margin:0;">A. 店頭（対面・即現金化）・宅配（自宅から送る）・出張（自宅で査定）から選べます。輸送事故や送付後の不当な減額を避けたいなら、高額品は店頭が安全。自宅から出したくない場合は宅配や出張も選択肢です。</p>
