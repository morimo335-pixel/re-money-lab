<!-- ===== H2-1（捨てると損する理由・感情＋市場データ） ===== -->
<h2>卒業した教科書・実家の本、捨てると損する理由</h2>

<!-- 💡 解決策（結論） -->
<div style="background:#FFF8E1;border-left:4px solid #C8923A;padding:18px 22px;margin:24px 0;border-radius:4px;">
<p style="margin:0 0 8px 0;"><strong style="color:#C8923A;">💡 解決策（結論）</strong></p>
<ul style="margin:0;padding-left:1.4em;line-height:1.85;">
<li><span style="color:#27AE60;">✅</span> <strong>専門書・医学書・赤本は、中古でも価値が落ちにくい</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>古紙に出すと、価値ある本まで「ただのゴミ」になる</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>売れば、次に学ぶ人の手に渡り、片付けの罪悪感も消える</strong></li>
</ul>
</div>

<p>「もう古いから」「書き込みがあるから」と、教科書や専門書を古紙回収に出してしまう方は多いです。でも現場の感覚で言うと、これは<strong>かなりもったいない</strong>。理由は3つあります。</p>

<p><strong>① 専門性の高い本は、中古でも値が落ちにくい</strong>。定価が1万円を超える医学書や、毎年新しい学生が必要とする大学の教科書は、需要が安定しています。電子書籍と違って紙の本は人に譲れる・売れるので、二次流通の市場が成立しているのです。実際、国内のリユース（中古品）市場は年3兆円規模まで伸びています（リサイクル通信調べ）。本もその一部です。</p>

<p><strong>② 古本チェーンの店頭では、専門書ほど不利になりやすい</strong>。街の大型古本店は、バーコード（ISBN）で機械的に査定する仕組みが中心です。すると、バーコードのない予備校テキストや、需要はあるけれど流通量が少ない専門書は、「査定対象外」や「一律安値」になりがち。<strong>専門店なら値がつく本が、総合店だと値がつかない</strong>——これが起こります。</p>

<p><strong>③ 捨てる罪悪感が、売ることで消える</strong>。子どもが夜遅くまで使った赤本、親が大切にしていた蔵書。これを「資源ゴミ」として出すのは、気持ちの面でもつらいものです。売って次の受験生や研究者の手に渡れば、<strong>「役立ててもらえた」</strong>という納得感が残ります。片付けが進まない一番の理由は、たいてい物量ではなく「捨てる踏ん切りがつかない」ことなんです。</p>

<div style="background:#E0F2F1;border:4px solid #00695C;border-radius:8px;padding:18px 22px;margin:28px 0;">
<p style="margin:0 0 10px 0;"><strong style="color:#004D40;">💡 業界人ワンポイント</strong></p>
<p style="margin:0;line-height:1.85;">現場でよく見るのが、<strong>「全部まとめて1社に出して、まとめて安く買われる」</strong>パターンです。雑多な本も専門書も一緒くたに総合店へ送ると、専門書の本当の価値が埋もれてしまう。逆に、<strong>専門書・医学書・赤本だけを"その道の専門店"に分けて出す</strong>ひと手間で、合計額がぐっと変わります。面倒に見えて、これがいちばん得をする送り方です。</p>
</div>

<h2>本の種類で「売る店」が変わる｜種類別ルーティング</h2>

<!-- 💡 解決策（結論） -->
<div style="background:#FFF8E1;border-left:4px solid #C8923A;padding:18px 22px;margin:24px 0;border-radius:4px;">
<p style="margin:0 0 8px 0;"><strong style="color:#C8923A;">💡 解決策（結論）</strong></p>
<ul style="margin:0;padding-left:1.4em;line-height:1.85;">
<li><span style="color:#27AE60;">✅</span> <strong>赤本・予備校テキスト → 受験本に強い専門店</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>大学教科書・専門書・医学書 → 学術書・医療書の専門店</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>小説・漫画・雑多な本 → まとめて扱う総合の宅配買取</strong></li>
</ul>

<p>本の買取店は、大きく<strong>「専門特化型」と「総合型」</strong>に分かれます。専門特化型は、特定ジャンルの相場と需要に精通していて、書き込みや記名があっても価値ある本なら減額しつつ買い取ってくれる。総合型は、幅広いジャンルを箱単位でスピーディーに引き取る代わりに、状態の基準が厳しめ——という住み分けです。下の表が、種類別の送り先の目安です。</p>

<!-- ===== H2-4（否認リスクと自衛術・確定率UP＝期待値コントロール） ===== -->
<h2>値がつかず損する人の失敗パターンと、5つの自衛術</h2>

<!-- 💡 解決策（結論） -->
<div style="background:#FFF8E1;border-left:4px solid #C8923A;padding:18px 22px;margin:24px 0;border-radius:4px;">
<p style="margin:0 0 8px 0;"><strong style="color:#C8923A;">💡 解決策（結論）</strong></p>
<ul style="margin:0;padding-left:1.4em;line-height:1.85;">
<li><span style="color:#27AE60;">✅</span> <strong>送料無料の最低冊数・金額を満たしてから送る</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>買取対象外の本（100均本・コンビニ漫画等）を混ぜない</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>査定を見て決めたいなら「自動承認」でなく承認コースに</strong></li>
</ul>
</div>

<p>実は、こうした本の宅配買取は、<strong>申し込んだ人のうち実際に買取が成立する割合（承認率）が、平均すると3〜4割ほど</strong>というデータがあります。裏を返せば、残りの6〜7割は「送り方のルール」を知らずに出して、思った値がつかなかったり買取不可になったり——ということ。逆に言えば、<strong>送る前のちょっとしたコツを押さえるだけで、あなたは"値がつく側"に回れます</strong>。ここから、つまずきやすい5つのパターンと、その自衛術をお伝えします。</p>

<p><strong>① 送料無料の条件を満たさず、少なく送る</strong>。専門書系3社は冊数（学参プラザは10点、専門書アカデミー・メディカルマイスターは5点）が目安、ブックサプライは査定額の合計が基準です。これを下回ると送料分が差し引かれることがあります。<strong>ある程度まとめてから送る</strong>のが鉄則です。</p>

<p><strong>② 買取対象外の本を混ぜる</strong>。100円ショップの本、コンビニ販売の廉価版漫画、付録、ひどい汚れや水濡れのある本などは、多くの店で買取不可です。特にブックサプライは状態基準が厳しめ。<strong>対象外の本は先に古紙回収へ回し、箱を無駄に重くしない</strong>のがコツです。</p>

<p><strong>③ 書き込み・記名の扱いを間違える</strong>。書き込みやマーカー、名前の記入がある本は、<strong>状態基準の厳しいブックサプライを避け、減額でも買い取ってくれる専門書系3社に送る</strong>のが正解。専門書系では、テキストの氏名欄を塗りつぶしたり切り取ったりしても査定に影響しないので、個人情報保護のうえでも安心です。</p>

<p><strong>④ 「気軽に送って、後で返してもらえばいい」と考える</strong>。査定結果に納得できずキャンセルすると、返送料は基本的に自己負担です。さらに、入金が早い<strong>「自動承認（スピード査定）」を選ぶと、査定と同時に売買成立でキャンセル不可</strong>。<strong>金額を見て決めたい人は、必ず「結果を承認してから売るコース」を選ぶ</strong>こと。宅配買取は基本「手放す前提」で使うサービスと割り切るのが安全です。</p>

<p><strong>⑤ 少額すぎて、買取が成立しないことがある</strong>。たとえばブックサプライは、まとまった冊数で<strong>合計3,000円以上</strong>になるような送り方が前提で、極端に少ない・安い本だけだと成立しないことがあります（沖縄・離島は対象外になる場合も）。<strong>漫画や小説は単品でなく、段ボールにまとめて</strong>送るのが向いています。目安のイメージとしては、<strong>状態のよい漫画セット10〜20巻分、または新しめの本を段ボール1箱しっかり詰めるくらい</strong>。数冊だけだと送料負けしやすいので注意してください。一方、専門書系3社は1冊1,000円以上の価値ある専門書なら少なめでも成立しやすい。<strong>「雑多な本はまとめて総合店、価値ある専門書は専門店」</strong>と覚えておけば、ムダ送りを防げます。</p>

<!-- 視覚要素③：送る前の自衛チェックリスト -->
<div style="background:#fff;border:3px solid #1B5E20;border-radius:10px;padding:20px 24px;margin:28px 0;">
<p style="margin:0 0 12px 0;font-weight:bold;color:#1B5E20;font-size:1.05em;">✅ 送る前に確認する5つのチェック</p>
<ul style="margin:0;padding-left:0;list-style:none;line-height:2.1;">

<!-- ===== H2-7 まとめ ===== -->
<h2>まとめ｜本は「種類別に分けて送る」だけで損しない</h2>

<!-- 💡 解決策（結論） -->
<div style="background:#FFF8E1;border-left:4px solid #C8923A;padding:18px 22px;margin:24px 0;border-radius:4px;">
<p style="margin:0 0 8px 0;"><strong style="color:#C8923A;">💡 解決策（結論）</strong></p>
<ul style="margin:0;padding-left:1.4em;line-height:1.85;">
<li><span style="color:#27AE60;">✅</span> <strong>①本のジャンルで売り先を分ける（専門書系／総合系）</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>②送料無料の条件と買取不可の本を先に確認する</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>③金額を見て決めたいなら承認コースを選ぶ</strong></li>
</ul>
</div>

<p>不要になった教科書や専門書は、「捨てる」と決めてしまう前に、ひと呼吸おいてみてください。やることはシンプルです。<strong>本のジャンルで売り先を分け、条件を確認してから送る</strong>。これだけで、古紙に出せばゼロだった本が、数千〜数万円に変わることがあります。今日からできるのは、次の3つです。</p>

<ol>
<li><strong>本をジャンルで仕分ける</strong>——赤本・予備校テキスト／大学教科書・専門書／医学書／一般書、の4つに分けるだけ。</li>
<li><strong>専門性の高い本を、専門店に寄せる</strong>——古本チェーンで安値になりがちな専門書こそ、専門の宅配買取で価値が出ます。</li>
<li><strong>送料無料の条件をそろえて送る</strong>——冊数・金額を満たし、対象外の本は抜く。金額を確認してから売りたいなら承認コースを選びましょう。</li>
</ol>

<p>大切なのは、<strong>「全部まとめて1社」で安く手放さないこと</strong>。種類別に分けるだけで、結果は大きく変わります。まずは無料査定で、あなたの本の価値を確かめるところから始めてください。</p>

<h2>よくある質問</h2>

<div style="margin:24px 0;">
<p style="margin:0 0 6px 0;font-weight:bold;color:#1B5E20;">Q. 書き込みやマーカーがある教科書でも売れますか？</p>
<p style="margin:0 0 18px 0;">A. 専門書系の専門店（学参プラザ・専門書アカデミー・メディカルマイスター）なら、元の価値が高い本は多少の書き込み・マーカーがあっても減額のうえ買い取ってくれます。一方、総合系のブックサプライは状態基準が厳しく、書き込みは買取不可になりやすいので、書き込みのある本は専門書系に送るのがおすすめです。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#1B5E20;">Q. 名前が書いてある参考書は売れませんか？</p>
<p style="margin:0 0 18px 0;">A. 売れます。専門書系3社は記名があっても買取可能で、テキストの氏名欄を塗りつぶしたり切り取ったりしても査定に影響しません。個人情報が気になる場合は、先に塗りつぶしてから送ると安心です。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#1B5E20;">Q. 何冊から送料無料になりますか？</p>
<p style="margin:0 0 18px 0;">A. 目安として、学参プラザは10点以上、専門書アカデミー・メディカルマイスターは5点以上で送料無料です。ブックサプライは査定額の合計が基準になります。冊数が少ないと送料分が差し引かれることがあるので、ある程度まとめてから送りましょう。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#1B5E20;">Q. 古い赤本や、何年も前の教科書でも値がつきますか？</p>
<p style="margin:0 0 18px 0;">A. 値がつくことがあります。専門書や定番の大学教科書は内容が古びにくく、古い年度の赤本も「過去の入試傾向を知りたい受験生」に需要があります。発行年だけで判断せず、まず査定に出してみるのがおすすめです。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#1B5E20;">Q. 査定額に納得できなかったら、返してもらえますか？</p>
<p style="margin:0 0 18px 0;">A. 返送は可能ですが、返送料は基本的に自己負担です。また、入金が早い「自動承認（スピード査定）」を選ぶと査定と同時に売買成立となり、キャンセルできません。金額を見てから決めたい場合は、必ず「結果を承認してから売るコース」を選びましょう。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#1B5E20;">Q. 小説や漫画もまとめて売りたいです。どこがいいですか？</p>
<p style="margin:0 0 18px 0;">A. 一般の小説・漫画・絵本・CD・DVDなど雑多なものは、幅広く扱うブックサプライが向いています。ただし状態基準が厳しめで、まとまった量・金額が前提になります（地域によっては対象外のことも）。状態のきれいな本を段ボールにまとめて送るのがコツです。専門書や赤本は、別途その専門店に分けて送ると高くなります。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#1B5E20;">Q. 実家の本を一気に片付けたいのですが、何から始めれば？</p>
<p style="margin:0;">A. まず「専門書・医学書・赤本」と「一般書」をざっくり2つに仕分けてください。価値が出やすい専門書系を専門店へ、残りの一般書を総合店へ送るだけで、安く手放すのを防げます。本以外にも片付けたい物が多い場合は、出張買取や遺品整理の業者と組み合わせると効率的です。</p>
</div>
