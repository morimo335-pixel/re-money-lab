<!-- ===== H2-3 IWC特有のコツ ===== -->
<h2>IWC査定の裏事情｜古くても・不動でも売れる「永久修理」の強み</h2>

<!-- 💡 解決策（結論） -->
<div style="background:#FFF8E1;border-left:4px solid #C8923A;padding:18px 22px;margin:24px 0;border-radius:4px;">
<p style="margin:0 0 8px 0;"><strong style="color:#C8923A;">💡 解決策（結論）</strong></p>
<ul style="margin:0;padding-left:1.4em;line-height:1.85;">
<li><span style="color:#27AE60;">✅</span> <strong>古い・サビ・不動でも「永久修理」で価値が残る（捨てない）</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>並行輸入品でも本社サービスは正規品と同じ＝減額理由にならない</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>国際保証書・余りコマ・純正尾錠は数万円の差になる</strong></li>
</ul>
</div>

<!-- 🛡️ IWC永久修理（記事の核を突出させる） -->
<div style="background:#F3EEE9;border:3px solid #5D4037;border-radius:10px;padding:18px 22px;margin:24px 0;">
<p style="margin:0 0 8px 0;font-weight:bold;color:#5D4037;font-size:1.05em;">🛡️ IWC最大の強み「永久修理」——だから古くても捨ててはいけない</p>
<p style="margin:0;line-height:1.85;">多くのブランドは生産終了から20〜30年で部品がなくなり正規修理を打ち切ります。でも<strong>IWCは「1868年の創業以来つくった全ての時計を永久に修理する」と公式に宣言している稀有なブランド</strong>。本社（シャフハウゼン）が数百万点の純正部品と全設計図を保管しているため、<strong>他店で「修理不可」と断られた錆び・不動・1950年代の個体でも復元できます</strong>。だから時計マニアの間では、こうした古いモデル（通称<strong>「オールドインター」</strong>）も驚くほどの高値で取引されます。中古市場では「再生のベース」として<strong>価値が100%残り、本体のみでも十数万〜数十万円</strong>に。<strong>汚いから・動かないからと捨てるのは絶対NG</strong>です。</p>
</div>

<p>この永久修理を前提に、残りの「査定の勘所」も3つ押さえておきましょう。</p>

<ul>
<li><strong>① 並行輸入品でも堂々と売れる</strong>：IWCの国際保証は非常にフェアで、<strong>正規店で買っても並行輸入店で買っても、本社の修理・アフターケアの料金に一切差をつけません</strong>。だから「並行輸入品だから」という理由で大きく減額してくる業者は信用に値しません。大手や一括査定で正当に評価してもらいましょう。</li>
<li><strong>② パイロットの革ベルトは「純正尾錠」を必ず残す</strong>：マークXVIII等の革ベルトは経年でボロボロになりがちですが、<strong>ベルト自体の劣化は軽微な減額</strong>。問題は<strong>IWC刻印の入った純正尾錠（バックル）が欠品していると数万円の減額</strong>になること。ちぎれたベルトでも、尾錠は外さずそのまま持ち込みましょう。</li>
<li><strong>③ 付属品で数万円変わる</strong>：<strong>国際保証書（ギャランティカード）の欠品は3万〜10万円以上の減額</strong>に。金属ブレスの<strong>余りコマ</strong>も1コマ1万〜3万円の差になります。引き出しの奥まで探して同封しましょう。</li>
</ul>

<p>古いIWCほど、真贋やパーツの価値を見抜ける鑑定士かどうかで査定が変わります。だから<strong>「ちゃんと説明してくれる店」を選ぶ価値が大きい</strong>。実際に「傷の理由や相場、永久修理の前提まで丁寧に教えてくれて納得できた」という声も多く、<strong>父や夫の形見を、納得して見送れるか</strong>は店選びで決まります。</p>

<p>高価なIWCを自宅で査定してもらう出張買取は、はじめてだと身構えるもの。<a href="/shucchou-kaitori-yobu-mae/" style="color:#5D4037;font-weight:600;">出張買取を呼ぶ前にやっておきたい準備とマナー</a>に目を通しておくと、当日落ち着いて立ち会えます。</p>

<p style="margin:0 0 18px 0;">A. 目的で選ぶのがおすすめです。1円でも高く売りたいなら複数社が競う一括査定のラクーダ、大手の安心感で失敗したくないならコメ兵、外出せず自宅で即現金化したいなら出張買取のウリエル（対応エリア限定）。いずれも査定・送料は無料なので、まず相場を確かめてから決められます。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#5D4037;">Q. 古くて動かない・サビたIWCでも売れますか？</p>
<p style="margin:0 0 18px 0;">A. 売れます。IWCは本社が創業以来の全ての時計を永久に修理する「永久修理制度」を持つため、他店で修理不可と断られた錆び・不動の個体でも、中古市場では「再生のベース」として価値が残ります。本体のみでも十数万〜数十万円になることも。汚いから・動かないからと捨てるのは厳禁です。まずプロの査定へ。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#5D4037;">Q. 並行輸入品のIWCでも売れますか？</p>
<p style="margin:0 0 18px 0;">A. 売れます。IWCの国際保証は正規店で買っても並行輸入店で買っても本社の修理・アフターケアに一切差をつけません。そのため「並行輸入品だから」と大きく減額してくる業者は信用に値しません。大手や一括査定で正当に評価してもらいましょう。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#5D4037;">Q. 箱や保証書がなくても売れますか？</p>
<p style="margin:0 0 18px 0;">A. 売れますが、減額はあります。国際保証書（ギャランティカード）の欠品は3万〜10万円以上下がることも。余りコマや純正尾錠の欠品も差になります。引き出しの奥まで探して、見つかったものは必ず同封してください。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#5D4037;">Q. IWCはオーバーホールしてから売った方が高い？</p>
<p style="margin:0;">A. いいえ、しない方が得です。IWCの正規オーバーホールは6万〜10万円以上と高額ですが、その費用ぶん査定が上がることはまずありません。買取店は自社の職人が業者価格で安く整備できるため、未オーバーホールの現状のまま出した方が、手元に残る現金は確実に多くなります。</p>
