<!-- ===== H2-2 京町家の搬出事情（京都差別化） ===== -->
<h2>なぜ京町家・旧家は費用が高くなりやすい？「うなぎの寝床」の搬出事情</h2>

<!-- 💡 解決策（結論） -->
<div style="background:#FFF8E1;border-left:4px solid #C8923A;padding:18px 22px;margin:24px 0;border-radius:4px;">
<p style="margin:0 0 8px 0;"><strong style="color:#C8923A;">💡 解決策（結論）</strong></p>
<ul style="margin:0;padding-left:1.4em;line-height:1.85;">
<li><span style="color:#27AE60;">✅</span> <strong>町家は奥に長く、中庭・狭い路地で「手運び割増」が出やすい</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>急で狭い階段・分解できない大型和家具で工数が倍増</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>現地見積もりで「横持ち料金」を最初に確認する</strong></li>

<p>京都、とくに上京・中京・下京・東山あたりの町家や旧家は、現代の家を前提にした基本料金では収まらない<strong>「京都ならではの搬出のしにくさ」</strong>があります。これが業者の見積もりに<strong>「横持ち料金（手運び割増）」</strong>として乗ってきます。</p>

<h2>京都ならでは｜仏壇じまい・お性根抜きの費用と進め方</h2>

<!-- 💡 解決策（結論） -->
<div style="background:#FFF8E1;border-left:4px solid #C8923A;padding:18px 22px;margin:24px 0;border-radius:4px;">
<p style="margin:0 0 8px 0;"><strong style="color:#C8923A;">💡 解決策（結論）</strong></p>
<ul style="margin:0;padding-left:1.4em;line-height:1.85;">
<li><span style="color:#27AE60;">✅</span> <strong>仏壇は処分前に「お性根抜き（魂抜き）」の法要が基本</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>菩提寺への法要はお布施1〜3万円＋御車代が目安</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>一括供養処分パックなら2万〜8万円程度が目安</strong></li>
</ul>
</div>

<p>京都は寺院・檀家の文化が濃く、実家を片付けるときほぼセットでついてくるのが<strong>「仏壇じまい（仏壇の処分）」</strong>です。京都では、仏壇を処分する前に<strong>「お性根抜き（おしょうねぬき）」</strong>——仏壇から本尊を移す法要を行うのが作法。京都に多い浄土真宗では「魂が宿る」とは考えないため、正しくは<strong>「遷仏法要（せんぶつほうよう）」</strong>と呼びます（一般には「魂抜き・閉眼供養」）。</p>

<p>費用の目安は、<strong>菩提寺の僧侶に法要を頼む場合でお布施1万〜3万円＋御車代5千〜1万円</strong>。お寺で本体を引き取りお焚き上げまで頼むと処分費1万〜8万円。<strong>魂抜きの手配から運搬・処分まで一括で頼むパック</strong>なら、合計2万〜8万円程度が目安です。お性根抜きを済ませた後なら、仏壇本体は<strong>京都市の粗大ごみ（数百〜数千円）</strong>で出すこともできます（道路際に出す心情的な抵抗はありますが）。極端に安い業者は供養を省いて勝手に処分するリスクがあるので、<strong>供養証明書を出してくれる業者</strong>を選ぶと安心です。</p>

<p>そしてもう一点。<strong>お性根抜きが済んだら、仏壇の引き出しの奥や隠し棚を必ず確認してください。</strong>古い金製のお鈴や仏具、記念硬貨が出てくることがよくあります。これらはゴミではなく"資産"。処分業者に渡す前に、<strong>貴金属としても査定してもらいましょう</strong>（上で紹介した骨董の査定とあわせて見てもらえます）。</p>

<h2>よくある質問</h2>

<div style="margin:24px 0;">
<p style="margin:0 0 6px 0;font-weight:bold;color:#6A4C9C;">Q. 京都の遺品整理、まず何から始めればいい？</p>
<p style="margin:0 0 18px 0;">A. ①貴重品（通帳・印鑑・権利証）と、茶道具・掛軸・着物・骨董など「売れるお宝」を先に自分で抜くことです。次に京都市の粗大ごみ・クリーンセンター持込で大型家具を出して総量を減らし、最後に残りを許可業者2〜3社に現地相見積もりで頼みます。この順番だと支払いがぐっと下がります。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#6A4C9C;">Q. 京町家は遺品整理が高くなりますか？</p>
<p style="margin:0 0 18px 0;">A. 高くなりやすいです。間口が狭く奥に深い「うなぎの寝床」構造や、細い路地でトラックを横付けできない立地が多く、長い手運び（横持ち）や急な階段作業で人件費が上乗せされます。一般的な住居より3万〜8万円ほど高くなることも。必ず現地を見せて、横持ち料金を含めた見積もりを取りましょう。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#6A4C9C;">Q. 悪質な業者を見分けるには？</p>
<p style="margin:0 0 18px 0;">A. 「一般廃棄物収集運搬業」の許可があるかを確認します。京都市の公式サイトの許可業者名簿で名前を照合できます。「無料で引き取る」とお宝を持ち去る業者、産業廃棄物や古物商の許可だけで家庭ごみを請け負う業者は避けましょう。現地見積もり＋書面を出すかも重要な判断材料です。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#6A4C9C;">Q. 仏壇の処分はどうすればいい？</p>
<p style="margin:0;">A. 京都では処分前に「お性根抜き（魂抜き・浄土真宗では遷仏法要）」を行うのが作法です。菩提寺への法要はお布施1〜3万円＋御車代が目安。魂抜き＋引き取り・処分の一括パックなら2万〜8万円程度。お性根抜きを済ませた後なら、仏壇本体は京都市の粗大ごみ（数百〜数千円）でも出せます。供養証明書を出す業者を選ぶと安心です。</p>
</div>
