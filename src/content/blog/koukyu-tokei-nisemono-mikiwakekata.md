<!-- ===== H2-1 素人判定の危険 ===== -->
<h2>高級時計の偽物の「見分け方」を、自分で試すのは危険です</h2>

<p>ネットで「高級時計 偽物 見分け方」と調べると、「ロゴのフォントを見る」「秒針の動きで判別」といった記事がたくさん出てきます。でも、現場の人間として正直に言います。<strong>その方法で素人が白黒つけるのは、今やほぼ不可能</strong>です。</p>

<p>理由は2つ。ひとつは、<strong>偽物の技術が上がりすぎた</strong>こと。昔は一目で分かる粗悪品が主流でしたが、今は内部に本物そっくりのムーブメント（機械）を積み、針の動きや文字盤まで本物に酷似した「スーパーコピー」が出回っています。本物との差はごくわずかで、<strong>本物を毎日触っている査定士でさえ「1本だけパッと見て即断」が難しい個体がある</strong>ほど。専用の10倍ルーペと比較見本があって、ようやく見抜けるレベルです。実際、財務省・税関が差し止めた偽ブランド品は1年で40万点超、その9割以上が高級ブランドの商標を悪用したコピー品で、<strong>ロレックスは何年も摘発額トップ</strong>です（出典：財務省 税関 知的財産侵害物品 差止状況）。それだけ精巧な偽物が大量に流通しているということです。</p>

<p>もうひとつ、もっと深刻なのが——<strong>自分で確かめようと、裏蓋を開けてはいけない</strong>ということ。</p>

<!-- 視覚①：自己分解のNGチェックリスト -->
<div style="background:#fff;border:3px solid #34418C;border-radius:10px;padding:20px 24px;margin:28px 0;">
<p style="margin:0 0 12px 0;font-weight:bold;color:#34418C;font-size:1.05em;">🚩 自分で裏蓋を開けると、こうなります</p>
<ul style="margin:0;padding-left:1.4em;line-height:2;">
<li><strong>❌ 修復不可の傷</strong>：専用工具なしでドライバー等でこじ開けると、金属ケースに深い傷が入り査定額が大きく下がる</li>
<li><strong>❌ 防水性能の崩壊</strong>：ゴムパッキンを噛んだり傷つけたりすると、本来の防水機能が完全に失われる</li>
<li><strong>❌ 精密部品の破損</strong>：一般家庭の埃や湿気が内部に入り、潤滑油の劣化や歯車の摩耗で動かなくなる</li>
</ul>
<p style="margin:12px 0 0 0;font-size:0.9em;color:#555;">＝ 本物だったとしても、自分で開けた時点で価値が落ちる。「確かめたい」気持ちが、いちばん損をします。</p>
</div>

<p>正直に言うと、<strong>真贋（本物か偽物か）の判断は、毎日何百点と本物を触っているプロでさえ、機械や経験を総動員してようやく下すもの</strong>です。手触り、刻印の彫りの深さ、金属の組成（店によってはX線で純度を測ります）——これらを総合して見ています。だからこそ、お金をかけずにこの目を借りられる「無料査定」を使わない手はないのです。</p>

<!-- ===== H2-2 恥ずかしさの解消 ===== -->
<h2>「偽物だったら恥ずかしい」は無用｜お店は"基準外"と返すだけ</h2>

<p>査定をためらう一番の理由が、<strong>「もし偽物だったら、店員さんに偽物だと言われて恥をかくのでは」「怒られるのでは」</strong>という不安だと思います。でも、現場のルールを知れば、その心配はいりません。</p>

<p>結論から言うと、<strong>買取店は、査定品が偽物だと思っても「これは偽物です」とはっきり言うことは、まずありません</strong>。なぜなら、メーカー以外の第三者が「これは偽物だ」と法的に断定することはできないから。だから店側は、こう言って静かにお返しします。</p>

<div style="background:#EDEFFA;border-left:5px solid #34418C;border-radius:0 8px 8px 0;padding:16px 20px;margin:24px 0;">
<p style="margin:0;line-height:1.85;font-size:1.02em;color:#34418C;"><strong>「申し訳ございません、こちらは当店の<u>買取基準外</u>となります」</strong></p>
</div>

<p>これだけです。頭ごなしに責められたり、その場で問い詰められたりすることは、基本的にありません。<strong>査定士は毎日のように真贋の難しい品を見ているので、特別なことだとは思っていません</strong>。むしろ「分からないからプロに見せに来た」のは、いちばん賢い行動だと受け止めています。</p>

<p>しかも知っておいてほしいのが、<strong>「基準外」と言われても、偽物とは限らない</strong>ということ。本物でも、次のような理由で「基準外（お取り扱いできません）」となることがあります。</p>

<ul>
<li>状態が極端に悪く、再販が難しいと判断された</li>
<li>手作りの個体差で、正規品の特徴と判別がつきにくかった（念のため見送られた）</li>
<li>シリアル番号が削れている等で、念のため取り扱いを控えられた</li>
</ul>

<p>だから「基準外」は「あなたが偽物を持ち込んだ恥ずかしい人」という意味では決してない。<strong>分からないまま査定に出すことは、何ひとつ恥ずかしくありません。</strong>むしろ堂々と「本物か分からないので見てほしい」と伝えれば大丈夫です。</p>

<!-- ===== H2-3 メルカリ出品の法的リスク（記事の核） ===== -->
<h2>本物か分からない時計、メルカリ出品はちょっと待って</h2>

<p>「店に行くのは面倒だし、メルカリやヤフオクに出してみよう」——これが<strong>いちばん危険な一手</strong>です。本物か分からない時計を、自分の判断で個人売買に出すのは、知らないうちに法律違反へ足を踏み入れることになりかねません。</p>

<!-- 🛡️ 法的リスク（記事の核を突出させる） -->
<div style="background:#F6ECEC;border:3px solid #B23A48;border-radius:10px;padding:18px 22px;margin:24px 0;">
<p style="margin:0 0 8px 0;font-weight:bold;color:#B23A48;font-size:1.05em;">🛡️ 知らずに偽物を出品すると「商標法違反」になる</p>
<p style="margin:0;line-height:1.85;">もしそれが偽物（コピー品）だった場合、フリマアプリに<strong>出品しただけで（売れる前でも）商標法違反</strong>に問われる可能性があります。個人であっても<strong>5年以下の拘禁刑、または500万円以下の罰金</strong>。実際に個人が逮捕された事例も多数あります。さらに、偽物と知って「本物です」と売れば詐欺罪（10年以下）にもなり得ます。<strong>「親の遺品で、偽物とは知らなかった」場合は刑事罰は免れやすい</strong>ものの、<strong>民事の責任は残ります</strong>——買った相手から「全額返金しろ」「損害を賠償しろ」と請求されれば、応じる義務があるのです（出典：商標法／刑法／弁護士各事務所の解説）。</p>
</div>

<p>怖い話に聞こえたかもしれません。もちろん、<strong>「偽物とは知らず、親の遺品としてただ保管していた」</strong>というような場合に、すぐ刑事事件になるわけではありません。ただ、真贋が分からないまま個人売買に出すと、後から「返金してほしい」といったトラブルに発展する可能性は残ります。そして、これを<strong>まるごと避ける方法は、とてもシンプル</strong>です。</p>

<p><strong>個人で売る前に、古物商の許可を持つプロの買取店に「無料査定」で見せること。</strong>これだけで、3つの安全が手に入ります。</p>

<ul>
<li><strong>① 本物なら</strong>、その場で適正な価格がつき、売却できる（遺産分割の話し合いも進む）</li>
<li><strong>② 偽物の疑いがあれば</strong>、多くの店では「基準外」として買取を見送られる＝<strong>知らずに違法な転売をしてしまうルートを、その手前で断ちやすくなる</strong></li>
<li><strong>③ 自宅から一歩も出ずに</strong>（宅配・出張査定なら）、これらが完結する</li>
</ul>

<p>つまり<strong>プロの無料査定は、あなたと家族を守る"安全なフィルター（盾）"</strong>。査定料もキャンセル料も無料の大手なら、リスクゼロで「本物か」「いくらか」だけを先に知れます。メルカリに出すのは、それを確かめてからでも遅くありません。</p>

<div style="background:#E0F2F1;border:4px solid #00695C;border-radius:8px;padding:18px 22px;margin:28px 0;">
<p style="margin:0 0 10px 0;"><strong style="color:#004D40;">💡 業界人ワンポイント</strong></p>
<p style="margin:0;line-height:1.85;">「鑑定だけしてほしい（売る気はない）」という依頼は、実は多くの買取店で断られます。お店の真贋確認は、あくまで"買い取るための査定"だからです。なので<strong>「売るかどうかは査定額を見て決めます」というスタンスで無料査定に出すのが、いちばんスムーズ</strong>。金額に納得できなければ、その場でキャンセルすればいいだけ。お金も手間もかかりません。</p>

<p>本物か分からない高級時計は、<strong>「分からないまま、触らず、プロに見せる」</strong>——これだけで、恥ずかしさも、損も、法律トラブルも、まとめて避けられます。今日からできるのは次の3つです。</p>

<!-- ===== よくある質問 ===== -->
<h2>よくある質問</h2>

<div style="margin:24px 0;">
<p style="margin:0 0 6px 0;font-weight:bold;color:#34418C;">Q. 偽物かもしれない時計を査定に出すのは恥ずかしくないですか？</p>
<p style="margin:0 0 18px 0;">A. まったく恥ずかしくありません。査定士は毎日のように真贋の難しい品を見ていて、慣れています。偽物だと思っても「偽物です」とは言わず、「買取基準外です」と返すだけで、責められることも通報されることもありません。「本物か分からないので見てほしい」と正直に伝えて大丈夫です。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#34418C;">Q. 本物か分からない時計を、メルカリやヤフオクに出してもいいですか？</p>
<p style="margin:0 0 18px 0;">A. おすすめしません。もし偽物だった場合、出品しただけで商標法違反（個人でも5年以下の拘禁刑または500万円以下の罰金）に問われる可能性があります。「知らなかった」場合も民事上の返金・賠償の責任は残ります。先にプロの無料査定で本物か確認すれば、こうしたリスクを丸ごと避けられます。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#34418C;">Q. 保証書がない・動かない時計でも売れますか？</p>
<p style="margin:0 0 18px 0;">A. 売れます。保証書を付ける法的義務はなく、本体のシリアルや刻印でプロは真贋を判断できます。動かない時計も提携工房で再生して再販できるため買取可能です。「書類がない・古い・動かない」は売れない理由にはなりません。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#34418C;">Q. 真贋（本物か）だけ確認してもらうことはできますか？</p>
<p style="margin:0 0 18px 0;">A. 「鑑定だけ（売る気はない）」は多くの店で断られます。お店の真贋確認は買取のための査定だからです。「査定額を見て売るか決めます」というスタンスで無料査定に出し、納得できなければキャンセルすればOK。料金はかかりません。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#34418C;">Q. ブランドが分からない時計でも査定できますか？</p>
<p style="margin:0;">A. できます。裏蓋やベルトのロゴ・型番を写真に撮って、宅配買取や一括査定に送れば、プロがブランド・真贋・相場を見極めてくれます。分からないまま、現状のまま出して問題ありません。</p>
</div>
