<h2>パネライを高く売るコツ｜ブロンゾは磨くと大損</h2>

<!-- 🛡️ ブロンゾ（記事の核を突出させる） -->
<div style="background:#F3ECE0;border:3px solid #6B4A2E;border-radius:10px;padding:18px 22px;margin:24px 0;">
<p style="margin:0 0 8px 0;font-weight:bold;color:#6B4A2E;font-size:1.05em;">🛡️ ブロンゾの「緑色のサビ」は磨いたら大損——10〜30万円減ることも</p>
<p style="margin:0;line-height:1.85;">パネライにはブロンズ（青銅）素材のモデルがあり、使ううちに表面に<strong>緑色のサビ（緑青・パティナ）</strong>が出ます。普通の時計なら錆びは大幅な減額要因ですが、<strong>ブロンゾに限っては、この緑青は減点どころか"そのケースだけの味"として加点される</strong>のがコレクターの世界。使い方で一本ずつ違う色になるため、同じものが二つとない価値になるのです。ところが「汚れだ」と思って市販のクリーナーでピカピカに磨いてしまうと、希少性が一瞬で消え、<strong>10万〜30万円規模で査定が落ちる</strong>取り返しのつかない損になります。<strong>ブロンゾは、絶対に磨かず・触らず、そのまま査定に出してください。</strong></p>
</div>

<p>この大前提を押さえたうえで、残りの「査定の勘所」も3つ見ておきましょう。</p>

<ul>
<li><strong>① 売る前のオーバーホールはしない</strong>：正規のオーバーホール（分解掃除）は10万円前後かかりますが、<strong>その費用ぶん査定が上がることはありません</strong>。買取店は提携工房で一般の半額以下で整備できるので、4万円かけて2万円上がっても赤字。<strong>動かなくても現状のまま、不具合を正直に伝えて出す</strong>のが手取り最大の鉄則です。</li>
<li><strong>② ボロボロのベルトも、純正の尾錠は捨てない</strong>：パネライはベルトを付け替えて楽しむ文化があるので、<strong>社外ベルトでも・革がボロボロでも本体は独立して評価</strong>されます。純正ベルトがないことの減額は実費相当（2〜4万円ほど）。ただし<strong>純正の尾錠・バックル</strong>はそれ自体に査定価値があるので、劣化したベルトごと捨てずに同封を。</li>
<li><strong>③ リューズガードのロックを確認</strong>：ルミノール特有の<strong>リューズガード（リューズを覆うレバー）</strong>は、ロックがしっかり閉まるか・ガタつきがないかを査定士が必ず見ます。不具合があると防水に関わるので減額（3〜5万円ほど）に。出す前に<strong>レバーがカチッと閉まるか</strong>だけ確かめておきましょう。</li>
</ul>

<p>もうひとつ。古いパネライほど、製造年ごとの細かな仕様差（文字盤やムーブメントの違い）でプレミアが変わります。だから<strong>「ちゃんと説明してくれる店」を選ぶ価値が大きい</strong>。形見や思い出の時計を手放すのは金額だけの話ではなく、<strong>納得して見送れたか</strong>が後の満足度を左右します。</p>

<p>とくに高価なパネライを自宅で査定してもらう出張買取は、はじめてだと身構えるもの。<a href="/shucchou-kaitori-yobu-mae/" style="color:#6B4A2E;font-weight:600;">出張買取を呼ぶ前にやっておきたい準備とマナー</a>に目を通しておくと、当日落ち着いて立ち会えます。</p>

<p>迷ったら、<strong>自宅から出さず複数社の最高値を引き出せるラクーダの無料一括査定</strong>から試すのが、いちばん失敗が少ない第一歩です。<strong>「もう流行ってないし二束三文かも」と思っていた時計が、思わぬ高値になることもあります。</strong>引き出しに眠ったパネライの価値を、まずは確かめてみてください。</p>

<h2>よくある質問</h2>

<p style="margin:0 0 6px 0;font-weight:bold;color:#6B4A2E;">Q. パネライを売るならどこがおすすめ？</p>
<p style="margin:0 0 18px 0;">A. 目的で選ぶのがおすすめです。1円でも高く売りたいなら複数社が競う一括査定のラクーダ、動かない・傷ありを安心して任せたいなら自社修理工房を持つコメ兵、外出せず自宅で即現金化したいなら出張買取のウリエル（対応エリア限定）。パネライは店による査定差が大きいブランドなので、1社で即決しないことが大切です。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#6B4A2E;">Q. ブロンゾ（ブロンズ）の緑色のサビは、磨いてから売るべき？</p>
<p style="margin:0 0 18px 0;">A. いいえ、絶対に磨かないでください。ブロンゾの緑青（パティナ）は減額要因ではなく、その個体だけの"味"としてコレクターに高く評価されます。市販クリーナーで磨き落とすと希少性が失われ、10万〜30万円規模で査定が下がることもあります。そのままの状態で査定に出すのが正解です。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#6B4A2E;">Q. 動かない・ベルトがボロボロのパネライでも売れる？</p>
<p style="margin:0 0 18px 0;">A. 売れます。パネライは部品需要・修復需要が世界的に高く、不動・本体のみでも人気モデルなら15万〜35万円の底値がつくのが特徴です。ベルトを付け替える文化があるため、社外ベルトでも・革がボロボロでも本体は独立して評価されます。純正の尾錠は捨てずに同封してください。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#6B4A2E;">Q. 箱や保証書がなくても売れる？</p>
<p style="margin:0;">A. 売れます。本体のシリアルや刻印でプロは真贋を判断できます。箱・保証書・余りコマ・専用工具がそろうと数万円〜十数万円プラスになるので探す価値はありますが、なくても買取は可能です。保証書に購入者名が書かれている場合、消したり切り取ったりしても査定額は下がりません。</p>
