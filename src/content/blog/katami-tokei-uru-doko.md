<!-- ===== H2-1 ===== -->
<h2>形見の時計は、しまい込む前に「どこで売るか」を決める</h2>

<!-- 💡 解決策（結論） -->
<div style="background:#FFF8E1;border-left:4px solid #C8923A;padding:18px 22px;margin:24px 0;border-radius:4px;">
<p style="margin:0 0 8px 0;"><strong style="color:#C8923A;">💡 解決策（結論）</strong></p>
<ul style="margin:0;padding-left:1.4em;line-height:1.85;">
<li><span style="color:#27AE60;">✅</span> <strong>引き出しで眠らせるほど、時計は機械的に劣化していく</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>まず無料査定で「いくらか・売れるか」を知るだけでも前に進む</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>査定＝即売却ではない｜価値を知ってから判断すればいい</strong></li>
</ul>
</div>

<p>形見の時計を前にして、多くの方が「いつか考えよう」と引き出しにしまい込みます。けれど、機械式の時計にとってそれは良くない選択です。<strong>暗い場所で動かさないまま放置すると、内部のオイルが固まり、サビが進んで機械の寿命が縮みます</strong>。気づいた頃には、本来つくはずだった価値が下がってしまうことも珍しくありません。</p>

<p>だからといって、いきなり売る決心をする必要はありません。大事なのは<strong>「まず価値を知る」</strong>こと。査定は無料の店が多く、<strong>査定を受けたからといって必ず売らなければいけないわけではありません</strong>。金額を見て、納得できなければ持ち帰ればいいだけです。「いくらになるのか」「そもそも売れるのか」を知るだけでも、止まっていた気持ちが少し前に進みます。</p>

<div style="background:#E0F2F1;border:4px solid #00695C;border-radius:8px;padding:18px 22px;margin:28px 0;">
<p style="margin:0 0 10px 0;"><strong style="color:#004D40;">💡 業界人ワンポイント</strong></p>
<p style="margin:0;line-height:1.85;">査定の現場で感じるのは、<strong>「価値が分かった瞬間に、ご家族の表情が変わる」</strong>ことです。「ただの古い時計」と思っていた物に思わぬ値がつくと、「だから父はこれを大事にしていたのか」と腑に落ちる。価値を知ることは、お金の話であると同時に、<strong>故人の思いを理解し直す機会</strong>でもあります。だからこそ、しまい込む前に一度、プロの目を通してみてください。</p>
</div>

<!-- ===== H2-2 ===== -->
<h2>親や家族の時計ほど、手放す判断がむずかしい理由</h2>

<!-- 💡 解決策（結論） -->
<div style="background:#FFF8E1;border-left:4px solid #C8923A;padding:18px 22px;margin:24px 0;border-radius:4px;">
<p style="margin:0 0 8px 0;"><strong style="color:#C8923A;">💡 解決策（結論）</strong></p>
<ul style="margin:0;padding-left:1.4em;line-height:1.85;">
<li><span style="color:#27AE60;">✅</span> <strong>形見を売る罪悪感は「物に魂が宿る」という日本人の感覚から来る</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>リユース＝捨てるのではなく「次の人にもう一度時を刻ませる」こと</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>手放したお金が、家族の新しい思い出に変わった例も多い</strong></li>
</ul>
</div>

<p>形見の時計を売るのに、なぜこんなにも気が重いのか。それは、<strong>多くの日本人が「遺品＝故人の魂の依り代」</strong>と感じているからです。「故人が大切にしていた物を換金するのは後ろめたい」「物にも魂が宿るのに、お金に変えていいのか」——こうした気持ちは、心が弱いからではなく、ごく自然なものです。</p>

<p>でも、ここで発想を変えてみてください。<strong>時計は、使われてこそ生きる道具</strong>です。引き出しの奥で止まり続けるより、その時計を本当に欲しい誰かのもとへ渡り、再び腕の上で時を刻む——それは「処分」ではなく、<strong>価値のバトンタッチ</strong>です。形見としての役割を十分に果たした時計を次の人へ手渡すことは、故人の所有物を再び社会で輝かせる、とても建設的な行為だと捉え直せます。</p>

<p>実際、終活や実家整理で形見の時計を手放した方が、その査定額を<strong>「孫と温泉やテーマパークに行く資金」</strong>に変えた、という声もあります。父の遺した時計が、家族の新しい笑顔と思い出に姿を変える。罪悪感だったものが、前向きな価値に昇華される瞬間です。「捨てる」でも「ただ売る」でもなく、<strong>「親が大切にした価値を、家族の未来へつなぐ」</strong>。この感覚が、手放す判断をぐっと楽にしてくれます。</p>

<!-- ===== H2-3 ===== -->
<h2>形見の時計は意外と高い？ブランド別の価値目安と「修理せず売る」鉄則</h2>

<!-- 💡 解決策（結論） -->

<p style="margin:0 0 8px 0;"><strong style="color:#C8923A;">💡 解決策（結論）</strong></p>
<ul style="margin:0;padding-left:1.4em;line-height:1.85;">
<li><span style="color:#27AE60;">✅</span> <strong>ロレックスなど人気ブランドは10年以上前の物でも高く売れる</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>動かない・箱や保証書がなくても買取は可能（減額はあっても断られない）</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>「動かないから」と自分で修理してから売ると、ほぼ赤字になる</strong></li>

<h2>「偽物扱い」「押し買い」が怖い人へ｜不安をなくす4つの知識</h2>

<!-- 💡 解決策（結論） -->

<li><span style="color:#27AE60;">✅</span> <strong>頼んでいない品の強引な勧誘（押し買い）は特定商取引法で違法</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>売った後でも8日間はクーリング・オフで取り戻せる</strong></li>
</ul>

<p>高級時計を売るとき特有の不安は、知識があればほとんど消せます。代表的な4つを順に解消しましょう。</p>

<p><strong>① 安く買い叩かれそう</strong>：最大の防御は<strong>相見積もり</strong>です。無料査定の店を複数選び、事前にLINE査定などで上限相場を把握しておく。現場で「他店では◯円でした」と伝えるだけで、業者はライバルを意識して適正額に近づけてきます。最初の提示額が異常に低いと感じたら、即決せず持ち帰るのが鉄則です。</p>

<p><strong>② 偽物だったら恥ずかしい・通報されそう</strong>：<strong>故意に偽物を本物と偽って売ろうとしない限り、知らずに持ち込んだだけで罪に問われることは一切ありません</strong>。鑑定料も無料で、万一の真贋結果が「対象外」でも、店側は「当社の基準では買取対象外です」と穏やかに伝えるだけ。気後れする必要はまったくありません。</p>

<p><strong>③ 強引に即決させられそう（押し買い）</strong>：訪問購入には<strong>特定商取引法</strong>が適用され、<strong>頼んでいない品の勧誘（不招請勧誘）は禁止</strong>。「時計の査定」で呼んだのに「金や貴金属はないか」としつこく居座る行為は違法です。毅然と「今日はありません」と断って構いません。</p>

<p>知っておきたいのが、<strong>万一、価値の高い遺品を無断で売却されても、売却から2年以内なら買い取った店に直接、返還を請求できる</strong>という法律（民法193条）。身内同士のフリクションに対する、法的なセーフティネットです。だからこそ、形見の時計は<strong>必ず当事者を交えて、合意のうえで査定に出す</strong>のが安心です。</p>

<p>そして、時計だけでなく実家全体の整理を控えているなら、<strong>片付けと同時に査定額を把握しておく</strong>と話がスムーズになります。出張買取なら時計も着物も家具もまとめて見てもらえ、「誰が何をいくらで」という客観的な数字が、兄弟・家族の話し合いの物差しになります。実家整理の進め方は、こちらの記事も参考にしてください。</p>

<p style="margin:0 0 10px 0;"><strong style="color:#004D40;">💡 業界人ワンポイント</strong></p>
<p style="margin:0;line-height:1.85;">査定の現場で実感するのは、<strong>「これは◯円です」という数字が出た瞬間、家族の会話が前向きになる</strong>ことです。「思い出だから」「もったいない」と平行線だったのが、客観的な金額という共通の物差しができると、急に冷静な相談に変わる。形見の時計を売る・残すの判断も、まずは家族で価値を共有することから始めると、後悔も揉め事も減ります。</p>

<p>形見の時計を手放すのは、寂しさや罪悪感を伴う作業です。でも、引き出しの奥で止まったままにするより、<strong>その時計を本当に必要とする誰かのもとで、もう一度時を刻ませる</strong>。それは、親が大切にした価値を未来へつなぐことでもあります。今日からできるのは、次の3つです。</p>

<ol>
<li><strong>まず無料査定で価値を知る</strong>——動かなくても、箱や保証書がなくても売れます。「いくらか」を知るだけで判断が進みます。修理してから売ると損なので、現状のまま出しましょう。</li>
<li><strong>1社で決めず、相見積もりか一括査定で比べる</strong>——人気ブランドほど業者で金額が変わります。自宅から出さず最高値を狙うなら一括査定が便利です。</li>
<li><strong>家族で合意して、安心して進める</strong>——古物商許可と相見積もりを確認。偽物の心配も、押し買いも、クーリング・オフという法律の盾があります。</li>
</ol>

<p>「売る」ことに罪悪感を抱く必要はありません。あなたが大切に査定に出した時計は、捨てられるのではなく、<strong>次の持ち主の腕の上で再び時を刻んでいきます</strong>。まずは、無料の査定を1本申し込んでみるところから始めてみてください。</p>

<h2>よくある質問</h2>

<div style="margin:24px 0;">
<p style="margin:0 0 6px 0;font-weight:bold;color:#6E2B3E;">Q. 動かない・電池切れの時計でも売れますか？</p>
<p style="margin:0 0 18px 0;">A. 売れます。買取店は自社や提携工房でメンテして再販できるため、止まっていても買取可能です。むしろ自分で修理してから売ると費用倒れで損するので、現状のまま査定に出すのが正解です。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#6E2B3E;">Q. 箱や保証書（ギャランティカード）がなくても大丈夫？</p>
<p style="margin:0 0 18px 0;">A. 大丈夫です。あれば査定額アップ要因ですが、保証書はメーカー再発行不可のため、なくても本体の刻印やムーブメントで真贋を見極められます。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#6E2B3E;">Q. もし偽物だったら、罪に問われますか？</p>
<p style="margin:0 0 18px 0;">A. 故意に偽物と知って本物と偽らない限り、知らずに持ち込んだだけで罪に問われることはありません。鑑定料も無料です。安心して査定に出せます。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#6E2B3E;">Q. 査定だけ受けて、売らなくてもいい？</p>
<p style="margin:0 0 18px 0;">A. はい。査定無料の業者なら、金額に納得できなければ売らずに断ってOKです。複数社で相見積もりを取り、一番高い1社で売るのが損しない方法です。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#6E2B3E;">Q. 自宅に他人を入れず、誰にも知られず売りたいのですが？</p>
<p style="margin:0 0 18px 0;">A. 宅配買取や一括査定なら、梱包して送るだけで完結し、自宅に人を入れる必要がありません。営業電話を避けたいなら、窓口が1社にまとまる一括査定が便利です。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#6E2B3E;">Q. 古いロレックスでも売れますか？</p>
<p style="margin:0 0 18px 0;">A. 売れます。ロレックスは中古需要が非常に高く、10年以上前のモデルでも高値がつきやすいブランドです。業者ごとに金額差が出やすいので、相見積もりや一括査定で比べると損しません。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#6E2B3E;">Q. 時計の相見積もりを取るのは失礼になりませんか？</p>
<p style="margin:0;">A. まったく失礼ではありません。複数社で査定を比べるのはごく普通のことで、買取店側も想定しています。むしろ1社で即決する方が、適正額を逃すリスクがあります。</p>
