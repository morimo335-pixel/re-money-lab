<p>ガラスが白くひび割れて何十年も放置されていたロレックスに15万円。ベゼル（外周のリング）が丸ごと紛失したサブマリーナに40万円。なぜこんな値段がつくのか——理由は3つあります。</p>

<ul>
<li><strong>① 部品（ドナー）としての需要</strong>：時計の中には数百の精密部品が入っています。高級ブランドは純正部品の供給が厳しく制限されていて、世界中の修理業者が「本物の部品」を探している。<strong>動かない時計は、他の時計を生き返らせる"ドナー"として価値がある</strong>のです。</li>
<li><strong>② 大手は自社で安く直せる</strong>：自社に修理工房を持つ買取店は、卸値に近いコストで分解掃除や部品交換ができます。直して再販できるから、壊れていても「仕入れ」として高く買えます。</li>
<li><strong>③ 海外への販路</strong>：円安の今、日本の中古時計とその部品は海外バイヤーにとって「世界で一番安く買える高品質な資源」。動かないムーブメント（機械部分）単体でも輸出されて取引されています。</li>
</ul>

<p>つまり「国内ではゴミに見えても、世界のどこかでは宝」。これが動かない時計が売れるカラクリです。</p>

<!-- ===== H2-3 修理は大損 ===== -->
<h2>修理してから売るのは大損｜唯一の例外は「安いクォーツの電池交換」</h2>

<p>「直してから持って行ったほうが高く売れるのでは？」——これが一番多い誤解です。数字で見てみましょう。</p>

<!-- 🛡️ 修理損得シミュレーション（記事の核を突出させる） -->
<div style="background:#F6EDE7;border:3px solid #B85C38;border-radius:10px;padding:18px 22px;margin:24px 0;">
<p style="margin:0 0 8px 0;font-weight:bold;color:#B85C38;font-size:1.05em;">🛡️ 「動けば8万円」の機械式時計、どっちが得？</p>
<p style="margin:0 0 10px 0;line-height:1.85;"><strong>A：自費で分解掃除（約6万円）してから売る</strong> → 8万円 − 6万円 ＝ 手元に残るのは<strong>2万円</strong><br>
<strong>B：何もせず、そのまま売る</strong> → 店は自社工房で約3万円で直せるので、その分だけ引いて買取 → 手元に<strong>5万円</strong></p>
<p style="margin:0;line-height:1.85;">結果、<strong>何もしないBのほうが3万円も多く残ります</strong>。買取店が引く「故障減額」は、あなたが払う高い修理代ではなく、<strong>店の安い修理原価</strong>を基準に計算されるからです。さらに、安く直そうと非正規の修理店で社外部品に交換すると「改造品」扱いになり、修理前より下がる・買取を断られることすらあります。<strong>動かないまま、不具合を正直に伝えて出す——これが鉄則です。</strong></p>
</div>

<p>ただし、<strong>唯一の例外</strong>があります。それは<strong>1〜5万円クラスの量産クォーツ時計の「電池交換」だけ</strong>。電池が切れたまま査定に出すと、店は「電池でなく内部の回路が壊れているかもしれない」という最悪のケースを想定して、1〜2万円大きめに引かざるを得ません。1,000〜2,000円で電池を換えて「動く」と証明できれば、この保険分の減額を防げて差し引きプラスになります。タグホイヤーなどのクォーツはこのパターンです。逆に、<strong>高級な機械式時計には触らないこと</strong>。これだけ覚えておけば大丈夫です。</p>

<!-- 💡 解決策（結論） -->
<div style="background:#FBF3EE;border-left:4px solid #B85C38;padding:18px 22px;margin:24px 0;border-radius:4px;">
<p style="margin:0 0 8px 0;"><strong style="color:#B85C38;">💡 解決策（結論）</strong></p>
<ul style="margin:0;padding-left:1.4em;line-height:1.85;">
<li><span style="color:#27AE60;">✅</span> <strong>高級ブランドの不動品 → 複数社が競う一括査定「ラクーダ」</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>1点を安全・確実に → 自社修理工房を持つ「コメ兵」</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>ノーブランド混じりの遺品ごと → 出張買取「ウリエル」（エリア限定）</strong></li>

<p>大事なのは、<strong>動かない時計は「出す店」で結果が極端に割れる</strong>ということ。修理設備のない総合リサイクル店に持ち込むと「動かない＝一律ジャンク」で数百円になることもあれば、部品需要を分かっている店なら数万円になる。同じ時計でこれだけ違います。だから店選びがすべてです。</p>

<p>「その店、本当に信用できるの？」と気になる方は、各社の実際の評判を先に確認しておくと安心です。</p>

<h2>よくある質問</h2>

<div style="margin:24px 0;">
<p style="margin:0 0 6px 0;font-weight:bold;color:#B85C38;">Q. 何十年も止まったままの時計でも、本当に売れますか？</p>
<p style="margin:0 0 18px 0;">A. ブランド時計なら売れる可能性が高いです。時計の中の精密部品には「他の時計を直すための部品（ドナー）」として世界中で需要があり、不動のロレックスに15万円、不動のオメガに約10万円がついた公開実績もあります。真贋が確認できない個体などを除けば、買取不可になるケースはごく一部です。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#B85C38;">Q. 修理や電池交換をしてから売ったほうが高くなりますか？</p>
<p style="margin:0 0 18px 0;">A. 基本的には損になります。買取店は自社や提携の工房で安く修理できるため、あなたが払う修理代ほど査定は上がりません。唯一の例外は1〜5万円クラスの量産クォーツ時計の電池交換（1,000〜2,000円）で、「動く」と証明できれば回路故障を疑う大きめの減額を防げます。高級な機械式時計には手を加えず、そのまま出してください。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#B85C38;">Q. ノーブランドや安い時計しかない場合はどうすればいいですか？</p>
<p style="margin:0;">A. ノーブランドの故障時計は値がつかないこともありますが、捨てる前に確認する価値はあります。遺品や実家整理で他の品物（着物・骨董・貴金属など）と一緒に出てきたなら、出張買取でまとめて見てもらうのが現実的です。時計単体で値がつかなくても、全体ではまとまった金額になるケースが多くあります。</p>
</div>
