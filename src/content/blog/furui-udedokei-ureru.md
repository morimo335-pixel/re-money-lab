<p style="margin:0;line-height:1.85;">保証書の代わりに、こんな紙きれが数万円の価値を持ちます。<strong>①メーカーの修理明細・国際サービス保証書</strong>（メーカー自身が「本物と認めて整備した」動かぬ証拠＝最強）、<strong>②オーバーホール証明書</strong>（再販時の整備コストが減るぶん査定に上乗せ）、<strong>③購入時の百貨店の領収書・レシート</strong>（真贋の疑いを払拭）。仏壇の引き出し・タンスの奥・書類ケースを30分探すだけで、査定額が変わることがあります。捨てる前に、紙ものこそ探してください。</p>
</div>

<p>もうひとつ大事なのが店選びです。正直に言うと、<strong>「保証書がないと買い取れません」と言う店は、紙の有無でしか本物を判断できない店</strong>。時計の目利きがいる店は、保証書がなくても本体だけで価値を弾き出します。古い時計が偽物扱いされたり、数百円のジャンク査定をされたりしたら、その店の知識不足を疑っていい——「本物か分からなくて不安」という方は、<a href="/koukyu-tokei-nisemono-mikiwakekata/" style="color:#5B6B3C;font-weight:600;">「偽物の見分け方」より大事な高級時計の売り方</a>も参考にしてください。</p>

<!-- ===== H2-4 売る前NG ===== -->
<h2>売る前にやってはいけないこと｜磨くと査定額が3〜5割下がります</h2>

<p>古い時計でいちばん多い失敗が、<strong>「きれいにしてから売ろう」とする善意の手入れ</strong>です。2026年のヴィンテージ市場は「ピカピカの美品」より「当時のまま」を高く買う、ノンポリッシュ（未研磨）至上主義。良かれと思った行動が、数十万円の価値を一瞬で消します。</p>

<p>古い腕時計は、「古いから」と諦めた人から順に損をするジャンルです。<strong>円安と世界的なヴィンテージ再評価で、2026年は古い時計にとって追い風の相場</strong>。一方で、引き出しの中では油が乾き、湿気でサビが静かに進んでいます。今日からできるのは次の3つです。</p>

<ol>
<li><strong>磨かない・直さない</strong>——きれいにした瞬間、ヴィンテージの価値が消えます。乾拭きだけでOK。</li>
<li><strong>紙ものを探す</strong>——保証書がなくても、修理明細やレシートが数万円の上乗せ材料になります。</li>
<li><strong>まず無料査定で「価値の有無」だけ知る</strong>——売るかどうかは金額を見てから。査定もキャンセルも無料です。</li>
</ol>

<p><strong>「ただの古時計」と思っていた一本が、世界のコレクターが探していた一本かもしれません。</strong>捨てる前に、まずは価値を確かめてみてください。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#5B6B3C;">Q. 何十年も前の古いセイコーやシチズンでも売れますか？</p>
<p style="margin:0 0 18px 0;">A. 型番によっては高く売れます。1960〜70年代のキングセイコーや初代グランドセイコーなどは、世界的なコレクターズアイテムとして再評価されていて、本体だけ・傷ありでも数万円の実例があります。「国産＝安物」という思い込みで捨てるのが一番の損です。まず無料査定で型番を見てもらってください。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#5B6B3C;">Q. 保証書も箱もありません。それでも買い取ってもらえますか？</p>
<p style="margin:0 0 18px 0;">A. 買い取ってもらえます。目利きのいる店は本体の型番・シリアル・内部の刻印で本物かどうかを判断できます。減額はブランドにより数千円〜10万円程度ありますが、メーカーの修理明細や購入時のレシートが見つかれば減額を抑えられます。「保証書がないと買取不可」と言う店は避けて、別の店に見せましょう。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#5B6B3C;">Q. 売る前にきれいに磨いた方が高くなりますか？</p>
<p style="margin:0;">A. 逆です。絶対に磨かないでください。今のヴィンテージ市場は「未研磨（ノンポリッシュ）・当時のまま」を最も高く評価するため、研磨剤で磨くと査定額が3〜5割下がることもあります。乾いた柔らかい布でホコリを拭く程度にとどめて、そのまま査定に出すのが正解です。</p>
