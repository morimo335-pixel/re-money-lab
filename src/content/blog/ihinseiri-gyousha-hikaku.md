<h2>遺品整理の業者選びで、いちばん多い失敗3パターン</h2>

<!-- ===== H2-2（悪徳業者の手口・恐怖→安心） ===== -->
<h2>悪徳業者の手口と、契約前に見抜くポイント</h2>

<!-- 💡 解決策（結論） -->
<div style="background:#FFF8E1;border-left:4px solid #C8923A;padding:18px 22px;margin:24px 0;border-radius:4px;">
<p style="margin:0 0 8px 0;"><strong style="color:#C8923A;">💡 解決策（結論）</strong></p>
<ul style="margin:0;padding-left:1.4em;line-height:1.85;">
<li><span style="color:#27AE60;">✅</span> <strong>「無料回収」のトラックや看板は、無許可の違法業者が多い</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>無許可業者に頼むと、不法投棄で"依頼主"が罰せられるリスク</strong></li>
<li><span style="color:#27AE60;">✅</span> <strong>頼んでいない買取の押し売り（押し買い）は違法・8日間取り戻せる</strong></li>
</ul>
</div>

<p>悪質な業者の手口は、知っていれば見抜けます。代表的な3つを押さえましょう。</p>

<p><strong>① 「無料回収」の罠</strong>：スピーカーで巡回するトラックや、空き地の「無料回収」看板。これらの多くは、自治体の<strong>一般廃棄物収集運搬の許可を持たない違法業者</strong>です。「無料」と言いながら、積み込み後に「運搬費・人件費は別」と数万円を請求してきます。</p>

<p><strong>② 不法投棄で"依頼主"が罰せられる</strong>：これが最も怖いポイント。無許可業者に渡した家財が山林などに不法投棄され、その中から故人の郵便物などが見つかると、<strong>捜査対象になるのは「ゴミを出した依頼主」</strong>です。撤去費用を負担させられるうえ、悪質と判断されれば廃棄物処理法違反（個人で5年以下の懲役または1,000万円以下の罰金）に問われるリスクすらあります。</p>

<p><strong>③ 押し買い（訪問購入）</strong>：「何でも高価買取」と言って上がり込み、売る気のない貴金属や着物を安く買い叩く手口。国民生活センターへの相談は年約8,000件で、<strong>被害の半数以上が70歳以上の女性</strong>です。ただし法律は消費者を守っています。<strong>頼んでいない買取の勧誘（不招請勧誘）は特定商取引法で禁止</strong>。万一売ってしまっても、<strong>8日間はクーリング・オフで取り戻せ</strong>ます。さらにこの期間中は、業者が「早く持って帰る」と言っても、<strong>「法律（引渡し拒絶権・特定商取引法58条の15）に基づき、8日間は品物を渡しません」と手元に留めておく権利</strong>が認められています。怖がらず、この盾を使ってください。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#37474F;">Q. 遺品整理の業者は、何社くらい見積もりを取ればいい？</p>
<p style="margin:0 0 18px 0;">A. 2〜3社が目安です。総額の安さだけでなく、見積書の明細の細かさ、許可の有無、追加料金なしの書面明記、担当者の説明の丁寧さを比べましょう。一括見積もりサービスを使うと、1回の入力で複数社を比較できて手間が省けます。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#37474F;">Q. 「無料回収」の業者に頼んでも大丈夫？</p>
<p style="margin:0 0 18px 0;">A. 注意が必要です。巡回トラックや空き地の「無料回収」看板の多くは、一般廃棄物の許可を持たない違法業者で、後から高額請求されたり、不法投棄で依頼主が責任を問われたりするリスクがあります。必ず許可の有無を確認しましょう。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#37474F;">Q. 遺品整理の費用を安くする方法は？</p>
<p style="margin:0 0 18px 0;">A. 一番効果的なのは「買取」です。骨董・着物・時計・貴金属などは、処分でなく買取に回すと、その額が作業費から相殺され総額が下がります。さらに自分で分別を進め、自治体の無料回収を活用すると、業者の作業量が減って費用を抑えられます。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#37474F;">Q. 買取と遺品整理は、別々に頼んだ方がいい？</p>
<p style="margin:0 0 18px 0;">A. 価値ある物が多そうなら、先に買取専門で査定を入れるのがおすすめです。遺品整理業者に一括で任せると、価値ある物まで安く引き取られたり処分されたりしがち。先にプロの目を通すと、総額が大きく変わります。買取も兼ねる遺品整理業者なら1社で完結できます。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#37474F;">Q. 女性一人で見積もりを受けても安全ですか？</p>
<p style="margin:0 0 18px 0;">A. 工夫すれば安全に進められます。女性スタッフ同行・指名プランを使い、玄関先で「今日は相見積もり中で即決しない」と先に伝えましょう。立ち会い不要（鍵預かり＋写真報告）のプランもあります。日中の時間帯に設定し、可能なら第三者に同席してもらうとより安心です。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#37474F;">Q. 遺品整理の業者選びは全国どこでも頼める？</p>
<p style="margin:0 0 18px 0;">A. 一括見積もりサービスや全国対応の業者を使えば、地方の実家でも依頼できます。地域によって対応業者や処分手数料が変わるので、まずは無料見積もりで「対応できる業者と相場」を確認するのが確実です。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#37474F;">Q. 契約後にキャンセルしたら、お金はかかりますか？</p>
<p style="margin:0 0 18px 0;">A. 業者によります。作業前でも高額なキャンセル料を設定する業者があるため、契約前に「解約条件・違約金」を必ず確認しましょう。即決を迫る業者ほどこの条件が不利なことが多いので、その場で署名せず持ち帰るのが安全です。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#37474F;">Q. 遺品整理は何日くらいかかりますか？</p>
<p style="margin:0 0 18px 0;">A. 多くは1日で完了します。1R〜1Kなら数時間、3LDK以上の一戸建てでも1〜2日が目安です。物量が多い・ゴミ屋敷状態・特殊清掃が必要な場合は延びることがあります。退去期限がある場合は、見積もり時に必ず作業日数を確認しましょう。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#37474F;">Q. ゴミ屋敷状態でも対応してもらえますか？</p>
<p style="margin:0 0 18px 0;">A. 対応できます。ただし通常より人員・トラック・時間がかかるため費用は上がり、悪臭がある場合は特殊清掃（消臭）も加わります。複数社で見積もりを取り、追加料金の条件を書面で確認しておくと安心です。</p>

<p style="margin:0 0 6px 0;font-weight:bold;color:#37474F;">Q. 「遺品整理士」がいる業者を選ぶべきですか？</p>
