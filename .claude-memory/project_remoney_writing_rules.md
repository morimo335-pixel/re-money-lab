## 🆕🆕🆕 2026-06-08 確定：リベ大「信頼される記事」3点セット（明日から全記事に・Satoshi指示）
**① 他者口コミを毎記事・出典付きで盛り込む**（自分の体験談だけでなく第三者の声＝信頼性↑＋「他で調べなくていい」読者メリット）。**🔒🔒2026-06-11 Satoshi確定（73「いい感じ！！」→74で運用強化指示）＝口コミ5原則**：
1. **その記事のテーマ・ブランド特有の口コミを調べる**（「ラクーダ良かった」の汎用でなく「古いセイコーを売ったら」「チューダーを査定したら」等、品目×業者の文脈口コミを優先。リサーチ段階で「[ブランド/品目] 売った 口コミ」「[業者名] [品目] 評判」で能動的に探す）
2. **毎記事10個**（本音2-3＋売った人の声7-8）
3. **1社最低3つ**（不満/注意系も混ぜる・出典URD付き）
4. **コピペ感をなくす**＝記事間重複禁止（同源口コミは差し替え or 賛否サマリ化・直近記事の💬ブロックと突合）＋記事内でも文型・長さをバラす
5. **AI感をなくす**＝綺麗なレビュー調禁止（「納得感が違いました」NG）→雑な実感の言い切り・ぼやき・具体物の名指し（「最初の店より8万高かったのでそのまま頼んだ」型）。長短・テンションを混在させる。業者の強い悪評（破損事故/高圧的対応等）は中傷リスクで不使用・ソフト系注意口コミで正直さを担保。**🚨口コミの記事間重複禁止（2026-06-11 Satoshi指摘・記事74）**：3社の口コミ元プールは小さい（みん評ラクーダ6件・ウリエル=公式+ヒカカクのみ）ため、同じ元口コミの別要約が複数記事に混ざりやすい。**新記事の口コミ確定前に、同クラスター直近記事の💬ブロックとgrepで突合**し、同源の口コミは「別の声に差し替え」or「総合評価のサマリ化（例:ヒカカク2.67点と賛否）」で回避。re-money-lab版の口コミ元＝**買取3社(ラクーダ/コメ兵/ウリエル)の利用者の声→みん評/買取系比較/Googleマップ/X**、**品目を売った人の本音→Yahoo知恵袋/SNS**（Amazon/楽天は物販レビューなので補助）。各記事に「3社の利用者の声1-2個＋品目売却の本音2-3個」を出典付きで。Deep Researchの口コミ項目を強化＋記事テンプレに「💬実際に売った人の声」ブロック。⚠️法務4鉄則＝出典明記・要約引用(丸ごと転載NG)・業者の悪評を断定/中傷NG・「No.1/最高」強調は景表法注意。
　**🆕2026-06-13 改定＝口コミ件数の上限を記事形式で分ける（無言でルールを破らない）**：①**複数業者比較記事**（パネライ3社等）＝従来どおり「1社3件まで・全体10件程度」。②**単独業者軸の記事**（78ブランド出張＝ウリエル単独等の新形式）＝**良い声4〜5件＋注意（厳しい声）2件まで＝計6〜7件上限**。理由＝単独軸で10件並べると量で押すセールスページ臭が出て逆に信頼が下がる（79本目レビューで確定）。良4＋注意2の6件が最も信頼が増す目安。
**② メリット・デメリット両方書く（デメリット先＋必ずフォロー）**：良いことだけの記事は読者が不信感→**デメリットを正直に先に書き、直後にフォロー**で納得させる（例「ラクーダは結果まで数日かかる＝デメリット。但しその分競争で高値が出やすい」）。読者は失敗回避でデメリットを先に知りたい→離脱減＋CV増。各買取業者(3社)にメリット＋デメリット(先)＋フォローを。
**③「どんな人におすすめか」を明確化**：メリデメ後に"改めて誰向けか"を断言→読者に「まさに自分だ」と思わせる。書く項目＝**読者の悩み＋理想の未来(売却後どうなるか)＝ベネフィット再提示**。re-money-labの既存「タイプ別診断(高く/安心/手間なし)」「向く人/向かない人」を、悩み＋理想の未来まで踏み込んで強化。

## 🆕🆕🆕 2026-06-08 確定：AI臭をもう一段消す（リベ大レビュー2本が指摘・全記事適用）
量産記事は構造が綺麗すぎて"AI量産臭"が出る。以下で人間味を出す：
- **①💡解決策(結論)BOXを全H2に置かない**→**冒頭結論・比較前(H2-2)・まとめの3箇所程度に絞る**(毎H2に出すと機械テンプレ臭の最大要因)。H2-1/特有/NG等のBOXは削ってよい。
- **②💬他者口コミは"綺麗なレビュー調"を避けリアルに**：「納得感が違いました」等の典型AIレビュー文はNG→雑な実感の言い切りに（例「最初の店より8万円高かったのでそのままお願いした」「父の時計で不安だったが説明が丁寧で安心できた」）。1-2件でも可。
- **③文章が綺麗すぎ・3行ごと綺麗着地・数字過多＝機械臭**→**現場の温度の一言**を混ぜる（例「正直、ブライトリングは店によって査定がかなり割れます」）。主役の数字を1つに絞り他は補足に。
- **④H2見出しはSEOワード型**（「○○の買取相場は？」「○○買取おすすめ3社比較｜どこが高い？」「高く売るコツ｜OHはNG」等、検索語を見出しに入れる）。
- **⑤CTA前に損得の実感を1行**（※鉄則②=具体客例の創作は禁止→「近所1社と複数社比較で十数万円違うことは珍しくない」等の一般化表現で）。
- **⑥3社送客はラクーダ偏りを避ける**→コメ兵の強み（状態の良い美品は自社販売力で一括査定より高いことも）を1行添えて中立感を出す。

## 🔒 ルール①：H2並び順の鉄則【全記事必須】

**Why:** 2026-05-06 リベ大添削で記事20に「ペルソナがカメラを売りたい人なら、読者が知りたいことを最初に持ってきた方が読者ファーストになる（相場・高く売る方法・売る先）」と指摘。

**How to apply:** 全記事のH2並び順は以下を必須とする。

### 必須H2並び順（売却系・キラー記事の標準）
1. **H2-1：相場（いくら？）** — 読者が真っ先に知りたい数字を冒頭で出す
2. **H2-2：高く売るコツ（どう売る？）** ＋ 末尾にミニCTA（上CTA）
3. **H2-3：売り先の選択肢（どこに売る？）** — 5択など並列で提示
4. **H2-4：おすすめ業者（mybest風フルカード）** ＋ フルA8 CTA（中CTA）
5. **H2-5：状態別の対応**（カビ・故障・ジャンク・古いetc）
6. **H2-6〜：業界トレンド・歴史・市場動向**（深掘りしたい人向け・後半）
7. **H2-N：まとめ** ＋ 下CTA（下A8カード）

### 配置ルール
- **冒頭4本 = 行動に直結する情報**（相場→売り方→売り先→業者）
- **後半 = トレンド・歴史・業界解説**
- **リード文末に1行誘導**：「※業界トレンド・市場動向は記事後半で詳しく」
- **A8 CTAは「上（H2-2末尾ミニ）／中（H2-4業者カード）／下（まとめ前）」の3点配置堅守**
- 内部リンクカード3枚は記事末尾（まとめ後）に固める

### 🚫 NG構成
- H2-1で「ブーム終了？」「時代遅れ？」など業界トレンド解説から入る → 読者離脱

### Phase 2チェック項目
- [ ] H2-1〜H2-4が「相場→売り方→売り先→業者」順
- [ ] 業界トレンド・歴史解説が記事後半（H2-5以降）に配置
- [ ] リード文末に「業界トレンドは後半で」誘導文あり

## 🔒 ルール②：1記事1焦点の鉄則【全記事必須】

**Why:** 2026-05-06 リベ大添削で記事20に「ブーム（世間の注目度）の話と高く売る話が混在していて、話の焦点が分かりづらい」と指摘。1記事に2つのテーマを併走させると読者離脱。お店で例えると「カメラ買取コーナー」と「カメラ博物館」の同居状態＝集中して買い物できない。

**How to apply:**
- **メインテーマは「読者が何を解決したいか」1つに絞る**
- **メインテーマと違う話題（業界トレンド・歴史・市場動向）は最大1H2のコラムに圧縮**
- **「Aの話とBの話」が並行する構造はNG** — 末尾に「【業界人コラム】」型でB話題を1本まとめる
- **タイトルKWに業界トレンド系（ブーム終了・時代遅れ等）を入れた場合**：本文メインは「売る話」、トレンドは末尾コラムで結論統合

### 焦点ぼやけの典型パターン
| 散漫構造（NG） | 焦点絞り（OK） |
|---|---|
| H2-1「ブーム終了の理由」H2-2「ブームではない根拠」H2-3「相場」H2-4「売り方」… | H2-1「相場」H2-2「売り方」…H2-N「【業界人コラム】ブーム終了は誤解」 |

### 違反履歴
- **2026-05-06 記事20**：旧構造でH2-1（終了理由）H2-2（終了根拠）が冒頭2本→ブーム関連2H2を【業界人コラム】1本に統合

## 🆕 ルール⑧：リード文の書き方（リベ大流）【2026-05-07 確定】

### 必須4要素（順序固定）：①想定読者の悩み（1人に強く刺さる具体悩み）→②解決策（結論）→③権威性（資格・実績・体験）→④メリット・ベネフィット（知識＋実利益／明るい未来）
- 例：①「父の遺品レコード、捨てるのも忍びないし業者も怖くて手が止まっていませんか？」②「実は古いレコードの大半は捨てる前に査定すれば数千〜数万円で売れます」③「買取営業8年・3万点超査定の私が…」④「5つの処分方法と高く売るコツが分かり、罠に引っかからず納得して手放せます」
- **リード文本体は目安500字（1分以内）**。長すぎは離脱。
- 違反NG：悩みが抽象的「不用品処分にお困りの方へ」／結論後回し「まず歴史を解説」／権威性なし／ベネフィットが「分かります」だけ。
